# LoginApp
A React Native Login App built with TypeScript featuring user authentication UI, input validation, screen navigation, and loading indicator. Designed as a beginner-friendly mobile application project.
# Login App (React Native)

## Features
- Login screen with validation
- Navigation between screens
- Loading indicator
- Clean UI design

## Tech Stack
- React Native
- TypeScript
- React Navigation

## How to Run
npm install
npx react-native run-android
